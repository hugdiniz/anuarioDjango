AnuárioUFRRJ: 
=============
Sistema de cadastro das unidades e sub-unidades organizacionais (depto., pro-retoria, setor administrativo, etc.) da universidade e cadastro dos seus respectivos servidores.
<br>Projeto desenvolvido em Django.
